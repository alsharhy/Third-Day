جدول وسوم HTML

الوسم الاستخدام خاصية متعلقة به
h1 يستخدم للعناوين الرئيسية align - محاذاة النص (قديم)
p يستخدم لإنشاء فقرة نصية dir - تحديد اتجاه النص
a يستخدم لإنشاء رابط href - تحديد عنوان الرابط
img يستخدم لعرض الصور src - تحديد مسار الصورة
table يستخدم لإنشاء جدول border - تحديد حدود الجدول
ul يستخدم لإنشاء قائمة غير مرتبة type - نوع الرموز (قديم)
ol يستخدم لإنشاء قائمة مرتبة start - تحديد بداية الترقيم
input يستخدم لإنشاء حقل إدخال type - نوع الإدخال
button يستخدم لإنشاء زر onclick - تشغيل كود JavaScript عند الضغط
